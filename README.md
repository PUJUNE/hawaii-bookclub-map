# 하와이 독서모임 지도

하와이 독서모임의 2021–2026년 모임 기록을 지도 위에 시각화한 페이지. 장소가 기록된 모임을 회차 번호 노드로 표시하고, 모임 순서를 화살표 엣지로 연결함.

**지도 페이지**: https://pujune.github.io/hawaii-bookclub-map/

## 구성

- `index.html` — 인터랙티브 지도 (Leaflet 기반 단일 파일, 빌드 불필요)
- `하와이 독서모임_책_저자_장소.xlsx` — 원본 모임 기록 (시기·책 제목·저자·모임날짜·장소)

## 사용법

- 지도 노드 클릭 → 해당 회차의 책·저자·날짜·장소 팝업
- 왼쪽 목록 클릭 → 해당 모임 위치로 이동
- 수도권 보기 / 전체 보기 전환, 이동 경로 표시 켜고 끄기
- 노드 색상은 연도별 구분 (2022–2026)
