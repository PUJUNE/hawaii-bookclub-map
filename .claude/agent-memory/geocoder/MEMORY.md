# geocoder 메모리

## 형식 규칙
- 좌표는 `[lat, lng]` 소수 4자리.
- index.html은 `lat > 36.5`를 수도권/전체 보기 분류 기준으로 사용 — 지방 모임(광주 등)은 '전체 보기'에서만 보인다.
- 같은 장소 재방문이 잦다. **아래 사전에 있으면 재검색 없이 그 좌표를 재사용**한다.

## 메모
- 같은 장소가 두 번 나올 때 좌표가 미세하게 다른 경우가 있음(과천과학관, 국립중앙박물관).
  둘 다 유효 범위 — 새 재방문 시엔 가장 최근 좌표를 쓴다.
- 좌표를 못 찍는 모임(온라인, 경조사)은 좌표 없이 보고하고, meeting-sync가 ll 필드를 생략하게 한다.
